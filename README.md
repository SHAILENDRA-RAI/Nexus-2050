# Techfest Nexus 2050 🚀
https://shailendra-rai.github.io/Techfest-IITB-Nexus-2050/

A futuristic 3D interactive web experience created for Techfest IIT Bombay.

## Features
- 3D cyber universe
- Interactive animations
- Futuristic UI/UX
- Custom cursor
- Particle effects
- Smooth transitions
- Responsive design

## Tech Stack
- React
- Vite
- Three.js
- Tailwind CSS
- Framer Motion

## Concept
A journey through the future of AI, Robotics, Space Technology and Human Innovation.
